Changelog
Added - Quests will send a message to a player when they damage, break, destroy, place, or kill.
				Example: break a block and it will say Block Destroyed 1/10
Added - You can now make kill quests with a players's name *Must be exact in capitalization.*
Added - Should be able to use all creatures up to Minecraft 1.3.2
Added - New Quest Type Shear for shearing sheep quests.
Added - New Quest Type Fish for fishing.
Added - New Quest Type Bucket_Fill for filling buckets.
Added - New Quest Type Plant for planting seeds.
Added - New Quest Type Till for tilling grass and dirt.
Added - New Quest Type Switch for switching levers, buttons, wooden doors, and trap doors.
Fixed - Gather, Block Break, Block Damage, and Block Place Quests now recognize durability of an item. *Should be able to use all items now.*
Fixed - Quest info will no longer show more then what you need for a quest.
				Example: Sheep Killed 20/10 will now show Sheep Killed 10/10 and no more.
Fixed - Quests objectives now multiply with Quest Levels.
Fixed - kill quests now recognize Slimes, Magmacubes, and EnderDragon
Fixed - Questioner was conflicting with /q so set it up to just not have /q if there is a coflict and made help reflect that.

Commands:
/uquest - lets you access most commands.
Aliases - /quest or /q (/q doesn't allways work if you have plugins like Questioner)

Standard Commands:
/uquest give - Gives quester a random quest.
/uquest done - Completes quester's current quest if done.
/uquest info - Gives quester info on their current quest.
/uquest stats - Gives quester their current stats.
/uquest amount - Tells the quester the amount of loaded quests.
/uquest drop - Drops quest.
/uquest top # - Shows quester the top questers.

Admin Commands:
/reloadquests - Reloads the quests.yml file.
/reloadconfig - Reloads config.yml (Deosn't seam to be working, will look at it later.)
/uquest set # - Set your quest to the number provided. *Added for troubleshooting*

Permissions:
uQuest.* - Gives you all uQuest commands. *Admin commands too*
uQuest.CanQuest - gives you acces to the uquest command but only lets you do help.
uQuest.CanQuest.* - Gives you acces to all standard uQuest commands.
uQuest.CanQuest.give - /uquest give
uQuest.CanQuest.info - /uquest info
uQuest.CanQuest.stats - /uquest stats
uQuest.CanQuest.done - /uquest done
uQuest.CanQuest.amount - /uquest amount
uQuest.CanQuest.CanDropQuest - /uquest drop
uQuest.admin - /uquest set,/reloadquests,/reloadconfig
